# xPUBench

This repository contains the code for four applications presentend in xPUBench. 

- fastclick-gpu: modified FastClick for hybrid CPU-GPU elements, following ROI and Z-C models.
- doca-gpu-app: GPU-only implementation based on an NVIDIA GPUNetIO sample.
- dpa-app: DPA application to run on RISC-V cores of the Bluefield-3.
- fastclick-dpu: modified FastClick to run on ARM cores of the Bluefield-2 and Bluefield-3 DPUs.
